# To-Do List App

This is a simple To-Do List app built with:
- **Backend**: FastAPI (Python)
- **Frontend**: Vue.js + Vuetify

## Features
- Add, update, and delete tasks.
- Mark tasks as completed.

## How to Run
1. Clone the repository.
2. Run the backend:
   bash
   ```
   cd backend
   uvicorn main:app --reload
   ```

   ```
   cd frontend
   npm install
   npm run serve
   ```
